A cross refrencing program written in c++11. 
Program runs in terminal or commandline. Takes a txt file and return an alphbetical
list of word contained in the file with corresponding line number it is found as well
as a count of the number of times appeared on that line.

Output:

word 			: <line number found at> : <number of occurances> , <line number found at> : <number of occurances> ,
