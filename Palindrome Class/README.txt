Palidrome Class

This is c++11 class that must be constructed with a string.
The object will split the screen into potiential strings and insert
them into a vector.The single public function called longest will return the longest palindrome as string.

Limitations:
characters that lower case and those that are upper case are not equal,
The class will not recognize that a sting "raCecAr" as a palidrome.
